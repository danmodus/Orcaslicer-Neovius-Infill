# OrcaSlicer with Neovius Infill

This is a fork of [OrcaSlicer](https://github.com/SoftFever/OrcaSlicer) that integrated the Neovius Triply Periodic Minimal Surface (TPMS) infill pattern.

## Features
- **Neovius Infill**: High-performance minimal surface infill with improved mechanical properties and periodicity.
